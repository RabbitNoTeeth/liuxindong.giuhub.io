# 1 数据的类型

**按照几何特征分类：**

- 点
- 线
- 面
- 体



**按照数据组织结构分类：**

- 矢量数据
- 栅格数据（由像素点组成）



# 2 数据的格式

- xxx.shp 

  矢量数据，一个完整的shp文件包含四部分：shp（ shape 形状文件）、prj（projection 投影文件）、dbf（data base file 数据库文件）、shx（shape index 索引文件）。

- xxx.tif

  栅格数据

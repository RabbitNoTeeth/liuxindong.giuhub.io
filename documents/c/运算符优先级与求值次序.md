**运算符的优先级与结合性**

| 运算符                                                      | 结合性   |
| ----------------------------------------------------------- | -------- |
| ()    []    ->    .                                         | 从左至右 |
| !     ~    ++    --    +    -    *    &    (type)    sizeof | 从左至右 |
| *    /    %                                                 | 从左至右 |
| +    -                                                      | 从左至右 |
| <<    >>                                                    | 从左至右 |
| <    <=    >    >=                                          | 从左至右 |
| ==    !=                                                    | 从左至右 |
| &                                                           | 从左至右 |
| ^                                                           | 从左至右 |
| \|                                                          | 从左至右 |
| &&                                                          | 从左至右 |
| \|\|                                                        | 从左至右 |
| ?:                                                          | 从左至右 |
| =    +=    -=    *=    /=    %=    &=                       | 从左至右 |
| ^=    \|=    <<=    >>=    ,                                | 从左至右 |

*注意：一元运算符 + 、- 、& 与 * 比相应的二元运算符 + 、- 、& 与 * 的优先级高。
